Embedded Learning Laboratories

Level 0 - Introduction to Algorithms with Scratch

Begin your programming journey by learning how computers follow instructions. Using Scratch, a fun and visual coding platform, you'll create simple interactive projects while exploring core concepts like sequences, loops, and decisions. Perfect for first-time coders.

Level 1 - Intermediate Scratch Programming

Build on your Scratch skills to design more complex projects. Learn to organize code using functions, apply advanced logic, and break problems into smaller, manageable steps. This level bridges the gap between beginner-friendly coding and more structured programming.

Level 2 - Arduino Basics: Coding Meets Electronics

Discover the world of physical computing. Learn how to program an Arduino to control LEDs, buttons, and basic sensors. This hands-on introduction shows how code can interact with the real world, making learning both exciting and practical.

Level 3 - Advanced Arduino Projects

Take your Arduino skills to the next level with projects that integrate multiple components and more sophisticated code. You'll deepen your understanding of electronics, improve your problem-solving skills, and learn how to troubleshoot and refine your creations.

Level 4 - Graphic Design Essentials

Master the basics of visual communication. Learn to create professional-looking designs by combining color theory, typography, and layout principles. Work with industry-standard tools to produce striking visuals for both digital and print use.
